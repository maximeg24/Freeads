You are on a laravel virtualhost server working on version 5.8.

To launch it and that it works without problem you will need :

-First of all you have to install composer in the location of the laravel folder.

-Launch wamp server

-launch powershell and write: 'cd (the position of your laravel)'. Then 'cd (blog)' and to finish to launch the server: 'php artisan serve'

-Now you just have to go to your browser and write :http://127.0.0.1:8000
